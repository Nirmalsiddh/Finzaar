# Finzaar

Finzaar is a full-stack stock trading platform built using the MERN stack. The application allows users to create an account, log in securely using JWT authentication, explore stock information, and access a dedicated dashboard for managing their account.

## Live Demo

Frontend: https://finzaar.vercel.app

Dashboard: https://finzaar-dashboard.vercel.app

Backend API: https://finzaar.onrender.com/

---

## Features

- User registration and login using JWT authentication
- Protected routes
- Responsive user interface
- Separate dashboard application
- REST APIs built with Express.js
- MongoDB database integration
- Stock data integration
- Clean and modular project structure

## Getting Started

### Clone the repository

```bash
git clone https://github.com/Nirmalsiddh/Finzaar.git
cd Finzaar
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```

Open the application in your browser:

```text
http://localhost:3000
```
